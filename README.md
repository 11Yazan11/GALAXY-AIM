# README

## **English**

### How to Run the Server and Join as a Client

1. **Clone the Repository:**
   First, open a command prompt window. Then, clone the repository to your local machine using the following command:
   ```bash
   git clone https://github.com/11Yazan11/GALAXY-AIM.git

   pip install -r requirements.txt

   ```
   Then, start the server:
   ```bash
   python server.py
   ```
   
   Now, make sure to open a new cmd window while keeping the other one with the server running, and past the following code:

   ```bash
   python public/client.py
   ```
