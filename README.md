### How to Run the Server and Join as a Client

1. **Clone the Repository:**
   Open command prompt.
   Clone the repository to your local machine using the following command:
   ```bash
   git clone https://github.com/your-repo-url.git

   pip install -r requirements.txt

   ```
   Then, start the server:
   ```bash
   python server.py
   ```
   
   Now make sure to open a new cmd window while keeping the other one with the server running.

   ```bash
   python client.py
   ```
